# Student Performance Prediction using Machine Learning

## Project Overview
This project predicts student academic performance using Machine Learning techniques. The system analyzes student information such as study hours, attendance, and marks to estimate overall academic performance.

## Features
- Student performance prediction
- Data preprocessing
- Simple data analysis
- Prediction based on sample dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
